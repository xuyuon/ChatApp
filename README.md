# CSCI3100 Project

ChatApp is a web-based chat application. It contains basic features that enable online messaging. 


## How to run

* Run the server (backend)
```
cd backend
```
Install the npm modules if you have not do so:
```
npm install
```
Start the server:
```
npm run dev
```

* Front end
```
cd frontend
```
npm install
```
npm start
```
